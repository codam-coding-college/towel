# Agent Index

## Snapshot
- Runtime CLI modules are split by concern: `user/local/share/towel/lib/run`, `user/local/share/towel/lib/install`, `user/local/share/towel/lib/export`.
- `towel install` prompts once for host export, supports `-y|--yes`, and treats non-interactive runs without `--yes` as user error.
- Export wrappers are generated by towel directly and stored in `~/.local/share/towel/exports/{bin,applications}`; host-visible entries are symlinks in `~/.local/bin` and `~/.local/share/applications`.
