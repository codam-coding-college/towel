# Changelog

All notable changes to this project will be documented in this file.
The format is based on [Keep a Changelog](https://keepachangelog.com/),
and this project adheres to a pre-1.0 [Semantic Versioning](https://semver.org/) contract:
`feat:` bumps minor, `fix:` bumps patch, and breaking changes bump minor (not major)
until `1.0.0`.
